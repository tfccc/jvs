### 一.线程和进程
1. 线程是进程的组成单位
2. 进程由至少一个线程构成
* java本身不能开线程, 是由native方法通过c++开启


### 二.并行和并发
1.[并行]: 多个线程同时运行
2.[并发]: 多线程操作同个资源


### 三.锁是什么, 如何判断锁的对象(见T05)
* 1.锁的对象-->调用者-->[1.实例对象, 2.模板]
* 2.一个对象只有一把锁, 一个Class也只有一把锁
* 3.先拿到锁的先执行
* 4.实例方法锁实例本身, 静态方法锁Class
* 5.未加锁的方法不会参与锁竞争


### 四.volatile
* 轻量级的同步机制
1. 保证可见性
2. 不保证原子性(不可分割的)
3. 禁止指令重排
