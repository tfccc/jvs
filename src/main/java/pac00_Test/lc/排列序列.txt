以数字的排列关系作为切入点, 推导公式: 算出每一个数字的索引, 用索引取到对应的数字, 然后remove掉已用的数字, 下次循环重复该过程.

以n=5为例, 循环中每一位的计算过程如下:
    num1 --> (k-1) / (n-1)! % (n-0) = idx1,  num1 = list[idx1],  list.remove(idx1);
    num2 --> (k-1) / (n-2)! % (n-1) = idx2,  num2 = list[idx2],  list.remove(idx2);
    num3 --> (k-1) / (n-3)! % (n-2) = idx3,  num3 = list[idx3],  list.remove(idx3);
    num4 --> (k-1) / (n-4)! % (n-3) = idx4,  num4 = list[idx4],  list.remove(idx4);
    num5 --> (k-1) / (n-5)! % (n-4) = idx5,  num5 = list[idx4]. (也可以直接list(0), 因为只剩最后一个数了)
    最终结果 = (num1 + num2 + num3 + num4 + num5)
其中:
    !代表阶乘
    / = java中的/
    % = java中的%
    n和k代表该题的输入参数
    list = [1,2,3,4,5]
    num1、num2... 表示结果的第一位、第二位数字
    idx1、idx2... 表示当前数字在list中的索引
